Dynamic Tracking and Object Manipulation

A 3-degrees of freedom robot arm was programmed with MATLAB to perform trajectories and sort bulbs according to their base size and colours. An additional camera was used for object, and centroid location detection. Setpoints for the trajectories were generated using quintic interpolation.

Mapping and Navigation with ROS

Programmed a Turtlebot3 Burger to map and navigate a maze using the attached LIDAR. A* path finding, and SLAM was used to navigate through the maze and detect the surrounding construction. The robot was programmed in Python and ROS and was able to simulate an arbitrarily constructed maze and navigate through it when instructed.

Playing Scrable with an ABB Robot

Programmed a 6-Degrees of Freedom ABB robot arm to perform various task such as palletizing. The final product was able to pickup and deposit Scrabble tiles on the game board with pre-determined words. This robot was programmed in RAPID and visualized in Robot Studio.
